
## Description

地址：https://github.com/nestjs/nest
文档：https://docs.nestjs.cn/

加密库：crypto-js

## 开始

```bash
$ npm install
$ npm run start
$ npm run start:dev
$ npm run start:prod

$ nest g controller name or path/name // 创建服务类
$ nest g service name or path/name    // 创建服务类
$ nest g module name or path/name     // 创建模块
```

## Test

```
$ npm run test

$ npm run test:e2e

$ npm run test:cov
```
