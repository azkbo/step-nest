## 操作

```bash
# 安装依赖库
$ npm install
# 运行项目
$ npm run start
# dev环境 运行项目 - 支持热更新
$ npm run start:dev
# prod环境 运行项目 - 支持热更新
$ npm run start:prod
# unit tests
$ npm run test
# e2e tests
$ npm run test:e2e
# test coverage
$ npm run test:cov
```
